Los patrones de diseño de software son técnicas para resolver problemas comunes en el desarrollo de software, un patrón de diseño resulta ser la solución de un problema de diseño. Los patrones se dividen en 3 grupos; Patrones de creacionales, patrones estructurales y patrones de comportamiento.
El presente documento establece las bases para el desarrollo del proyecto "PAPEXP", una aplicación en consola programada en C# diseñada para la gestión de inventario y punto de venta de la papelería Arcoíris. El objetivo principal es crear una herramienta que permita optimizar las operaciones diarias del negocio, incluyendo la administración de ventas, el control de inventario y la gestión de la información de los empleados.
A lo largo de este documento se detallarán los requerimientos funcionales y no funcionales, el diagrama de clase, la estructura del proyecto y la metodología usada que guiará el desarrollo de esta solución.
